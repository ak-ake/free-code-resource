C++ File Input/Output

Learn file operations in C++ using streams.

File Operations

#include <fstream> #include <iostream> // Write to file ofstream outfile("output.txt"); outfile << "Hello World" << endl; outfile.close(); // Read from file ifstream infile("output.txt"); string line; while (getline(infile, line)) { cout << line << endl; } infile.close();

File Modes

  • ios::in - Read mode
  • ios::out - Write mode
  • ios::app - Append mode
  • ios::binary - Binary mode
Best Practices:
  • Always check if file opened successfully
  • Close files after use
  • Handle file not found errors
  • Use binary mode for non-text files

Use Cases

  • Data persistence
  • Configuration files
  • Logging
  • Data processing
