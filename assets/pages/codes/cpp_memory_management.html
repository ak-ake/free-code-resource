C++ Memory Management

Master dynamic memory allocation and smart pointers.

Pointers

// Declare pointer int* ptr; // Get address int x = 10; ptr = &x; // Address of x // Dereference int value = *ptr; // Pointer to pointer int** ptr_ptr = &ptr;

Dynamic Memory

// Allocate memory int* arr = new int[10]; int* single = new int(5); // Use memory arr[0] = 100; *single = 20; // Free memory (IMPORTANT!) delete[] arr; delete single;

Smart Pointers

#include <memory> // unique_ptr - exclusive ownership unique_ptr<int> ptr1(new int(10)); unique_ptr<int> ptr2 = make_unique<int>(20); // Automatically deleted // shared_ptr - shared ownership shared_ptr<int> sp1 = make_shared<int>(30); shared_ptr<int> sp2 = sp1; // Deleted when last reference is destroyed
Best Practices:
  • Prefer smart pointers over raw pointers
  • Use make_unique and make_shared
  • Always delete allocated memory
  • Avoid memory leaks

Use Cases

  • System programming
  • Performance optimization
  • Resource management
