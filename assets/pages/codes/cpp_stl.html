C++ Standard Template Library (STL)

Master STL containers and algorithms for powerful C++ programming.

Vector - Dynamic Array

#include <vector> using namespace std; // Create vector vector<int> nums; nums.push_back(10); nums.push_back(20); // Access elements int first = nums[0]; int second = nums.at(1); // Remove elements nums.pop_back(); nums.erase(nums.begin());

Map - Key-Value Pairs

#include <map> // Map creation map<string, int> ages; ages["Alice"] = 25; ages["Bob"] = 30; // Access and iterate for (auto pair : ages) { cout << pair.first << ": " << pair.second << endl; }

Set - Unique Elements

#include <set> // Set creation set<int> unique_nums; unique_nums.insert(5); unique_nums.insert(3); unique_nums.insert(7); // Automatically sorted // Check existence if (unique_nums.find(5) != unique_nums.end()) { cout << "Found" << endl; }

STL Algorithms

#include <algorithm> // Sort sort(nums.begin(), nums.end()); // Reverse reverse(nums.begin(), nums.end()); // Find auto it = find(nums.begin(), nums.end(), 10); // Copy vector<int> copy_nums(nums); // Count int count = count(nums.begin(), nums.end(), 10);
Best Practices:
  • Use iterators for safe element access
  • Prefer STL containers over raw arrays
  • Use auto for cleaner code
  • Check iterator validity after modifications

Use Cases

  • System programming
  • Game development
  • Performance-critical applications
  • Embedded systems
  • High-performance computing
