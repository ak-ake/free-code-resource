CSS Grid Layout

Master CSS Grid for creating powerful, flexible 2D layouts.

What is CSS Grid?

CSS Grid is a 2D layout system that allows you to create complex layouts with rows and columns. It provides precise control over item placement and alignment.

Basic Grid Setup

.container { display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: 100px auto 100px; gap: 1rem; }

Responsive Grid Layout

.grid-container { display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); gap: 2rem; padding: 2rem; } .grid-item { background: #f0f0f0; padding: 1.5rem; border-radius: 8px; }

Key Grid Properties

Container Properties

  • grid-template-columns - Define column layout
  • grid-template-rows - Define row layout
  • gap - Space between grid items
  • justify-items - Horizontal alignment
  • align-items - Vertical alignment
  • grid-auto-flow - Item placement direction

Item Properties

  • grid-column - Column span
  • grid-row - Row span
  • justify-self - Individual horizontal alignment
  • align-self - Individual vertical alignment

Advanced Grid with Named Areas

.layout { display: grid; grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; grid-template-columns: 200px 1fr 1fr; gap: 1rem; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }

Grid Functions

/* repeat() - Repeat column/row patterns */ grid-template-columns: repeat(3, 1fr); /* minmax() - Set min and max sizes */ grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* fr unit - Fractional units */ grid-template-columns: 2fr 1fr 1fr;
Pro Tips:
  • Use fr units for flexible sizing
  • Use repeat() to reduce code repetition
  • Use auto-fit for responsive grids
  • Named grid areas improve readability

Use Cases

  • Website layouts (header, sidebar, footer)
  • Photo galleries and card layouts
  • Dashboard layouts
  • Magazine-style layouts
  • Responsive design systems
