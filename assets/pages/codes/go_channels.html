Go Channels

Synchronize and communicate between goroutines.

Basic Channels

package main import "fmt" func main() { // Create channel results := make(chan string) go func() { results <- "message" }() // Receive from channel msg := <-results fmt.Println(msg) }

Buffered Channels

package main import "fmt" func main() { // Buffered channel capacity 2 ch := make(chan int, 2) ch <- 1 ch <- 2 fmt.Println(<-ch) fmt.Println(<-ch) }

Select and Close

package main import "fmt" func main() { ch1 := make(chan string) ch2 := make(chan string) go func() { ch1 <- "one" }() go func() { ch2 <- "two" }() for i := 0; i < 2; i++ { select { case msg1 := <-ch1: fmt.Println("Channel 1:", msg1) case msg2 := <-ch2: fmt.Println("Channel 2:", msg2) } } }
  • Unbuffered channels - Synchronous communication
  • Buffered channels - Asynchronous with capacity
  • Select - Wait on multiple channels
  • Close - Signal completion
  • Range - Iterate over channels
Best Practices:
  • Close channels from sender only
  • Use select for multiplexing
  • Avoid channel deadlocks
  • Range over channels for iteration

Use Cases

  • Inter-goroutine communication
  • Worker pools
  • Fan-in/fan-out patterns
  • Rate limiting
