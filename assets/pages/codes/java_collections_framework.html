Java Collections Framework

Master Java collections for efficient data management.

List - Ordered Collection

import java.util.*; // ArrayList - Resizable array List<String> list = new ArrayList<>(); list.add("Apple"); list.add("Banana"); list.remove(0); // LinkedList - Efficient insertion/deletion List<Integer> linkedList = new LinkedList<>(); linkedList.add(1); linkedList.addFirst(0); linkedList.removeLast();

Set - Unique Elements

// HashSet - No order, fast lookup Set<String> hashSet = new HashSet<>(); hashSet.add("apple"); hashSet.add("apple"); // Won't add duplicate // TreeSet - Sorted, slower insertion Set<Integer> treeSet = new TreeSet<>(); treeSet.add(3); treeSet.add(1); treeSet.add(2); // Automatically sorted // Iterate through set for (int num : treeSet) { System.out.println(num); }

Map - Key-Value Pairs

// HashMap - Fast lookup by key Map<String, Integer> map = new HashMap<>(); map.put("apple", 5); map.put("banana", 3); // Access values int count = map.get("apple"); if (map.containsKey("orange")) { } // Iterate through map for (String key : map.keySet()) { System.out.println(key + ": " + map.get(key)); }

Complete Collections Example

import java.util.*; public class CollectionExample { public static void main(String[] args) { // List of students List<String> students = new ArrayList<>(); students.add("Alice"); students.add("Bob"); // Grades map Map<String, Double> grades = new HashMap<>(); grades.put("Alice", 95.5); grades.put("Bob", 87.3); // Print all for (String student : students) { System.out.println(student + ": " + grades.get(student)); } } }

Queue - FIFO Structure

// Queue interface (FIFO) Queue<String> queue = new LinkedList<>(); queue.add("First"); queue.add("Second"); String first = queue.remove(); // Removes first // Priority Queue PriorityQueue<Integer> pq = new PriorityQueue<>(); pq.add(5); pq.add(1); pq.add(3); System.out.println(pq.poll()); // Returns 1

Collection Methods

  • add/remove - Add or remove elements
  • size/isEmpty - Check collection size
  • contains - Check if element exists
  • clear - Remove all elements
  • sort - Sort collection
  • stream - Process collections functionally
Best Practices:
  • Choose right collection for your use case
  • Use generics for type safety
  • Know time complexity of operations
  • Use immutable collections when possible

Use Cases

  • Storing and retrieving data
  • Implementing caches
  • Processing streams of data
  • Graph and tree structures
  • Task scheduling and prioritization
