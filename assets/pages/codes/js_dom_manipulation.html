JavaScript DOM Manipulation

Learn how to select, modify, and interact with HTML elements using JavaScript.

Selecting Elements

// Select by ID const element = document.getElementById('myId'); // Select by class name const elements = document.getElementsByClassName('myClass'); // Select using CSS selectors const element = document.querySelector('.my-class'); const allElements = document.querySelectorAll('.my-class');

Modifying Content

// Change text content element.textContent = 'New text'; // Change HTML content element.innerHTML = '<p>New HTML</p>'; // Change attributes element.setAttribute('data-id', '123'); element.id = 'newId';

Complete DOM Manipulation Example

// Select the element const button = document.querySelector('.toggle-btn'); const content = document.querySelector('.content'); // Add event listener button.addEventListener('click', function() { // Toggle class content.classList.toggle('hidden'); // Change button text button.textContent = button.textContent === 'Show' ? 'Hide' : 'Show'; });

Creating and Removing Elements

// Create element const newDiv = document.createElement('div'); newDiv.textContent = 'Hello World'; // Add to DOM document.querySelector('body').appendChild(newDiv); // Remove element newDiv.remove();

Styling Elements

// Direct style modification element.style.backgroundColor = 'blue'; element.style.padding = '10px'; // Add/remove classes element.classList.add('active'); element.classList.remove('inactive'); element.classList.toggle('highlight');

Event Handling

// Click event element.addEventListener('click', function(event) { console.log('Clicked!'); }); // Multiple events element.addEventListener('mouseenter', function() { console.log('Mouse entered'); }); element.addEventListener('mouseleave', function() { console.log('Mouse left'); });

Common Events

  • click - Element clicked
  • mouseenter/mouseleave - Mouse enters/leaves
  • focus/blur - Element focused/unfocused
  • input/change - Input value changed
  • keydown/keyup - Keyboard events
  • submit - Form submitted
Best Practices:
  • Use querySelector for modern code
  • Avoid modifying HTML strings (use methods)
  • Use event delegation for dynamic elements
  • Always remove event listeners when done

Use Cases

  • Form validation and interaction
  • Toggling UI elements
  • Responsive navigation menus
  • Dynamic content updates
  • Interactive components
