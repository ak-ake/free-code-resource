JavaScript Event Handling

Master event handling to create interactive and responsive web applications.

Adding Event Listeners

// Basic event listener element.addEventListener('click', function(event) { console.log('Button clicked'); }); // Arrow function syntax element.addEventListener('click', (event) => { console.log('Button clicked'); });

Event Object Properties

element.addEventListener('click', (event) => { // Event properties console.log(event.target); // Element that triggered event console.log(event.type); // Event type ('click') console.log(event.clientX); // Mouse X position console.log(event.clientY); // Mouse Y position console.log(event.key); // Key pressed (for keyboard) // Prevent default behavior event.preventDefault(); // Stop event propagation event.stopPropagation(); });

Form Validation Example

const form = document.querySelector('.contact-form'); const emailInput = form.querySelector('input[type="email"]'); form.addEventListener('submit', (event) => { // Prevent form submission event.preventDefault(); // Validate email if (emailInput.value.includes('@')) { console.log('Email is valid'); form.submit(); } else { console.log('Invalid email'); emailInput.classList.add('error'); } });

Event Delegation

// Listen on parent for child clicks document.querySelector('.list').addEventListener('click', (event) => { if (event.target.classList.contains('item')) { console.log('Item clicked:', event.target.textContent); } });

Mouse Events

  • click - Mouse button clicked
  • dblclick - Double-clicked
  • mousedown - Mouse button pressed
  • mouseup - Mouse button released
  • mousemove - Mouse moving
  • mouseenter - Mouse enters element
  • mouseleave - Mouse leaves element

Keyboard Events

document.addEventListener('keydown', (event) => { if (event.key === 'Enter') { console.log('Enter key pressed'); } if (event.ctrlKey && event.key === 's') { event.preventDefault(); console.log('Ctrl+S pressed'); } });

Form Events

const input = document.querySelector('input'); // Input value changed input.addEventListener('input', (event) => { console.log('Value:', event.target.value); }); // Input focused input.addEventListener('focus', () => { input.classList.add('focused'); }); // Input blurred input.addEventListener('blur', () => { input.classList.remove('focused'); });
Best Practices:
  • Use event delegation for dynamic elements
  • Remove listeners when elements are removed
  • Use preventDefault() wisely
  • Consider performance with frequent events

Use Cases

  • Form submission and validation
  • User interactions (clicks, hovers)
  • Keyboard shortcuts
  • Real-time search/filtering
  • Drag and drop functionality
