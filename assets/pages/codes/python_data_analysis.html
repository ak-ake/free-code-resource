Python Data Analysis

Learn data analysis techniques using pandas, NumPy, and matplotlib.

Pandas Basics

import pandas as pd # Create DataFrame df = pd.DataFrame({ 'name': ['Alice', 'Bob', 'Charlie'], 'age': [25, 30, 35], 'city': ['New York', 'London', 'Paris'] }) # Read from CSV df = pd.read_csv('data.csv') # Display data print(df.head()) # First 5 rows print(df.describe()) # Statistics

Data Manipulation

# Filter data young = df[df['age'] < 30] # Select columns names = df['name'] subset = df[['name', 'age']] # Add new column df['is_adult'] = df['age'] >= 18 # Group by grouped = df.groupby('city').mean() # Sort data sorted_df = df.sort_values('age', ascending=False)

Data Analysis Workflow

import pandas as pd import numpy as np import matplotlib.pyplot as plt # 1. Load data df = pd.read_csv('sales.csv') # 2. Clean data df = df.dropna() # Remove missing values df['price'] = pd.to_numeric(df['price'], errors='coerce') # 3. Analyze data summary = df.describe() correlation = df.corr() # 4. Visualize plt.scatter(df['price'], df['quantity']) plt.show()
Best Practices:
  • Always inspect your data first
  • Handle missing values appropriately
  • Check for outliers and anomalies
  • Document your analysis process
  • Validate results with visualizations
