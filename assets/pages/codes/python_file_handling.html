Python File Handling

Master file operations in Python for reading, writing, and manipulating files.

Opening and Closing Files

# Using open() function file = open('file.txt', 'r') # 'r' = read mode content = file.read() file.close() # Using context manager (recommended) with open('file.txt', 'r') as file: content = file.read() # File automatically closes

File Modes

  • 'r' - Read (default)
  • 'w' - Write (overwrites)
  • 'a' - Append (adds to end)
  • 'x' - Create (fails if exists)
  • 'b' - Binary mode
  • 't' - Text mode (default)
  • '+' - Read and write

Reading Files

with open('data.txt', 'r') as file: # Read entire file content = file.read() # Read line by line for line in file: print(line.strip()) # Read specific lines file.seek(0) # Go to beginning first_line = file.readline() all_lines = file.readlines() # Returns list

Writing to Files

# Write to file (overwrites) with open('output.txt', 'w') as file: file.write('Hello World\n') file.write('Line 2\n') # Append to file with open('output.txt', 'a') as file: file.write('Line 3\n') # Write multiple lines with open('output.txt', 'w') as file: lines = ['Line 1\n', 'Line 2\n', 'Line 3\n'] file.writelines(lines)

Working with Different File Types

JSON Files

import json # Read JSON with open('data.json', 'r') as file: data = json.load(file) # Write JSON data = {'name': 'John', 'age': 30} with open('data.json', 'w') as file: json.dump(data, file, indent=2)

CSV Files

import csv # Read CSV with open('data.csv', 'r') as file: reader = csv.reader(file) for row in reader: print(row) # Write CSV with open('output.csv', 'w', newline='') as file: writer = csv.writer(file) writer.writerow(['Name', 'Age']) writer.writerow(['John', 30])

File Operations

import os # Check if file exists if os.path.exists('file.txt'): print('File exists') # Get file size size = os.path.getsize('file.txt') # Rename file os.rename('old.txt', 'new.txt') # Delete file os.remove('file.txt')
Best Practices:
  • Always use context managers (with statement)
  • Handle file not found errors
  • Use proper file encoding (UTF-8)
  • Close files properly even if error occurs

Use Cases

  • Reading configuration files
  • Processing data files
  • Logging application events
  • Data import/export
  • Batch file processing
