Ruby File Handling

Read, write, and manipulate files in Ruby.

Reading Files

# Read entire file content = File.read("file.txt") # Read line by line File.foreach("file.txt") { |line| puts line } # Using open File.open("file.txt") do |file| file.each_line { |line| puts line } end

Writing Files

# Write to file (overwrites) File.write("output.txt", "content") # Append to file File.write("output.txt", "\nmore content", mode: "a") # Using open with write mode File.open("output.txt", "w") do |file| file.puts "Hello" end

File Operations

# Check if file exists if File.exist?("file.txt") File.delete("file.txt") end # Copy file FileUtils.copy("source.txt", "dest.txt") # Directory operations Dir.mkdir("new_dir") Dir.glob("*.txt")
  • File.read() - Read entire file
  • File.write() - Write to file
  • File.open() - Open with block
  • File.foreach() - Iterate lines
  • Dir operations - Directory management
Best Practices:
  • Always use blocks for file operations
  • Files close automatically with blocks
  • Handle exceptions (IOError, FileNotFound)
  • Use absolute paths for clarity

Use Cases

  • Log files
  • Configuration loading
  • Data import/export
  • File processing
